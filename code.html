    if (Number.isInteger(startNumber) && Number.isInteger(endNumber)) {

//generate some numbers
function generateNumbers(start, end) {
    //generate numbers between start and end
    let numbers = [];


    for (let i = start; i < = end; i = i + 1) {
        // if i is less than or equal to a hundred it will do whatever we 
        // want it to in the curly bracket
        // write down the number i is on
        numbers.push(i);
    }

    return numbers;


}


//display the numbers on the page
function displayNumbers(numbers) {

}

How it works

The code is structured in three functions.

getValues()

we want to go to our site.js page and extract the id's from our input elements that are located in our HTML. Our input elements located in our HTML have an ID of startInput and endInput. keying in let startInput = document.getElementById ('startInput') in our site.js allows us to extract the elements from HTML. After successfully extracting our elements, we then want to create new variables i.e. let startNumber/let endNumber = a function we call parseInt. parseInt turns the string of text into an integer. So now our string of text startInput/endInput will turn into numbers once we run and debug our js code to make sure we arent making any errors. If numbers arent entered we will get a NAN on either the startNumber/endNumber. If numbers are entered we then move to the next step that will validate if the inputs are integers. If the startNumber/endNumber are both integers, js will generate an array of numbers using the function generateNumbers. If one of the startNumber/endNumber comes back false, js will then show a pop up alerting the user to enter a valid startNumber and endNumber.

generateNumbers()

Our next function is generateNumbers, once we know that both integers for startNumber/endNumber are true. generateNumbers will generate numbers between our startInput endInput to create an array of integers. The array of integers are based off of the loop function for (let i = start; i < = end; i = i + 1) numbers.push(i)}. This means that as long as the starting value is less than or equal to the ending value, the loop will continue looping while adding plus one to the starting value after every loop cycle until the starting value is no longer less than the ending value. Once the loop is finished, and the numbers are generated into an array [], that will complete the function generateNumbers. The generatedNumbers will be returned back to the function getValue, where they will now = the function generateNumbers(startNumber, endNumber).

displayNumbers()

This is our final function in getting the numbers to display on the page. we can control the table on the html by writing let tableHtml = '';. This means the function displayNumbers will use the table tag that has been written for our HTML. To get our table to display the array of numbers between the startNumber, and the endNumber. We must first run another loop function. for (let i = 0; i < numbers.length; i = i + 1). This means that the loop starts with i equaling 0 and as long as the i (index) is less than the numbers.length (numbers.length being the array), the loop will continue cycling and through each cycle the Loop will add one to the index until the index is no longer less than the numbers in the array. Since we will be showing our even numbers in bold print on our HTML table, we will have to go a step further. To do this we will need to make another function where currentNumber = number[index] and the className will be assigned 'even' or 'odd' depending on whether the current number of the array has any remainder left after dividing the number by 2 using the modulus operator. Once we have given a className to both the even and odd numbers that makes up the array, we will write HTML for that number, since its not one specific number we will use the function tableHtml = tableHtml + ` class=" ${className}"> ${currentNumber} `. After we create the table in our js, we place that table in our HTML by letting our table element in HTML be extracted by its ID through js.
